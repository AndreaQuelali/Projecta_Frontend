# ProjectaFrontend

Sistema de Gestión de Proyectos que permite planificar, organizar, asignar y dar seguimiento a proyectos mediante tableros, facilitando el control del avance, la colaboración entre usuarios y la trazabilidad de las tareas.

## Development server

Para iniciar el servidor de desarrollo, ejecuta:

```bash
ng serve
```

Una vez que el servidor esté en ejecución, abre tu navegador y navega a `http://localhost:4200/`. La aplicación se recargará automáticamente cada vez que modifiques alguno de los archivos de origen.
